# Exercise Database Expansion Plan

This document outlines the expansion of the exercise database with 2-3 variations for each exercise using different equipment (dumbbells, resistance bands, kettlebells).

## Current Exercise Count
- Strength: ~25 exercises
- Cardio: ~15 exercises  
- Flexibility: ~15 exercises
- Balance: ~10 exercises
- **Total: ~65 exercises**

## Target After Expansion
- **Target: ~200+ exercises** (3x expansion with variations)

## Expansion Strategy

For each existing exercise, we'll add:
1. **Dumbbell variation** (if applicable)
2. **Resistance band variation** (if applicable)
3. **Kettlebell variation** (if applicable)
4. **Bodyweight variation** (if not already bodyweight)
5. **Machine/cable variation** (for some exercises)

- Difficulty levels may vary between variations
